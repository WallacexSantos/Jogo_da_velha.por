# jogo_da_velha.por

Jogo da Velha em Portugol Studio

Este projeto é um Jogo da Velha desenvolvido utilizando o Portugol Studio.
O objetivo do jogo é proporcionar uma forma simples e interativa de praticar lógica de programação, estruturas condicionais, repetição e manipulação de matrizes.

O jogo permite que dois jogadores disputem partidas no terminal, realizando jogadas alternadas até que haja um vencedor ou empate.
